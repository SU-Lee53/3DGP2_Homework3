# 3DGP2_Homework3
3D Game Programming Homework 3 from TUKorea Game Engineering

2차 과제를 기반으로 다음 내용을 추가하여 구현한다.

 

1. 지형을 테셀레이션을 적용하여 LOD 기능을 구현한다. 

2. 플레이어가 빠르게 이동할 때 블러링 효과를 계산 쉐이더를 활용하여 구현한다.

3. 그림자 매핑을 적용하여 그림자를 구현한다. 

 

언급하지 않은 나머지 내용은 각자가 알아서 정하여 구현하고 설명 문서에 잘 서술한다.



Based on the second assignment, implement the following additional features:

Apply tessellation to the terrain and implement a LOD (Level of Detail) system.

When the player moves at high speed, implement a blurring effect using a compute shader.

Implement shadows using shadow mapping.

Any other details not explicitly mentioned should be defined and implemented as you see fit, and described clearly in the accompanying documentation.
